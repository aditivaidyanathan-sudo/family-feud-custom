# Family Feud — Custom Edition

A fully self-contained, browser-based Family Feud game built in a single HTML file. No installation, no server, and no external dependencies are required. The game is intended to be run by a host on a laptop or projected screen at parties, game nights, and similar events.

---

## Getting Started

1. Download `Family Feud Custom.html`
2. Open it in any modern browser
3. Read the rules, edit your questions, and play

No further setup is required.

---

## How It Works

**Step 1 — Read the Rules**
The landing page includes a full breakdown of how to play — expand the How to Play section to read the rules for the main game, the strike and steal mechanic, Fast Money, and how the winner is determined.

**Step 2 — Set Up Your Teams**
Select the number of teams (2–8), assign each a name and colour, and enter the number of players per team.

**Step 3 — Edit Your Questions**
Customise the 25 rounds and 5 Fast Money questions before the game begins. Question text, categories, point multipliers, answers, and point values are all editable. Questions can be added or removed entirely, and all content can be reset to the included defaults at any time.

**Step 4 — Play**
The host controls the board throughout. Tiles are clicked to reveal answers, strikes are logged for incorrect answers, and points are awarded to the winning team at the end of each round. After 3 strikes the board enters Steal mode, allowing any team to attempt to claim the points. The Rounds screen allows the host to jump to any round at any point during the game.

**Step 5 — Fast Money**
The two highest-scoring qualifying teams each send two players to answer 5 quick-fire questions against the clock. Scores from this round are added to the main scoreboard.

## Features

**Gameplay**
- Supports 2–8 teams with customisable names, colours, and player counts
- 25 rounds of questions with animated tile reveals
- Points multipliers per round — ×1, ×2, or ×3
- Strike system — three strikes triggers a steal opportunity open to all other teams
- Qualification bar tracking which teams are on course for Fast Money
- Fast Money finale in which the top 2 qualifying teams compete in a timed quick-fire round
- Final podium displaying a ranked scoreboard and winner announcement

**Question Editor**
- All questions can be fully customised before the game begins
- Rounds and Fast Money questions can be added or deleted freely
- Editable fields include question text, category, points multiplier, answer text, and point values
- Individual answers can be added or removed, with a minimum of 2 and a maximum of 8 per question
- A live points total is displayed per question and highlights in red if the 100-point limit is exceeded
- A validation prompt appears on launch if any question is over the limit, with the option to correct it or proceed regardless

**Design**
- Visual style consistent with the Family Feud aesthetic — navy, gold, and red colour palette
- SVG logo, scanline overlay, and flip tile animations
- Fully responsive layout suitable for a range of screen sizes

## Included Questions

The game includes 25 PG-13 rounds and 5 Fast Money questions across topics such as food, travel, school life, daily habits, animals, and technology. All content is family-friendly and appropriate for mixed-age groups. Every question can be replaced using the built-in editor prior to starting the game.

---

## Customisation Notes

- All questions can be replaced with a custom theme — films, sport, company trivia, a specific occasion, etc.
- Deleting questions in the editor allows for a shorter game
- Using ×2 and ×3 multipliers on selected rounds adds variety and tension
- Enabling GitHub Pages on the repository will generate a shareable live URL
